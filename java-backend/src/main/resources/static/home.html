Back End Documentation

Welcome to Ethan's social media project. This is the main page for the back end (built with Java).

The main purpose of the back end is to securely communicate with the database, save user information, validate users upon login, store post information, and ensure users can edit their posts.

To use the back end you'll want to point the front end applications to point to the following end points:

  • localhost:8080/login (POST)
  • localhost:8080/user/ (GET)
  • localhost:8080/user/register (POST)
  • localhost:8080/user/{id} (PUT, DELETE)
  • localhost:8080/post/ (GET)
  • localhost:8080/post/add (POST)
  • localhost:8080/post/delete (DELETE)
  • localhost:8080/post/{id} (PUT)
